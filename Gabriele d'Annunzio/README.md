# Gabriele d'Annunzio - progetto web

Sito web statico dedicato a Gabriele d'Annunzio, con focus su biografia, contesto storico, Carta del Carnaro e diritti della donna.

## Pagine

- `index.html` - home con collegamenti rapidi alle sezioni
- `biografia.html` - pagina biografia (struttura base)
- `contesto.html` - pagina contenuto storico con timeline e curiosita
- `carta.html` - pagina Carta del Carnaro (struttura base)
- `donne.html` - pagina diritti della donna (struttura base)

## Tecnologie

- HTML5
- CSS3 (`css/style.css`)
- JavaScript vanilla (`js/script.js`)
- Node.js per server locale (`js/server/server.js`)

## Funzionalita principali

- Tema chiaro/scuro con salvataggio in `localStorage`
- Navigazione tramite pulsanti nella home
- Scroll fluido verso sezioni interne
- Filtro della timeline nella pagina contesto
- Generatore casuale di "fatti curiosi"

## Avvio del progetto

### 1) Apertura diretta

Apri `index.html` nel browser.

### 2) Server locale Node.js (consigliato)

Da root progetto:

```bash
node js/server/server.js
```

Poi apri:

```text
http://127.0.0.1:3000
```
